# システム設計書

## はじめに

このサイトはMkDocs + Material for MkDocsで作成されたシステム設計書のサンプルです。
Mermaidを使用してシーケンス図やその他のダイアグラムを表示できます。

## 主な機能

- **認証システム** - OAuth2.0を使用したセキュアな認証
- **データ処理基盤** - 大規模データの効率的な処理
- **API連携** - 外部サービスとのシームレスな連携

## ドキュメント構成

### システム概要
システムのアーキテクチャと構成について説明します。

### シーケンス図
主要な処理フローをシーケンス図で可視化しています。

- [ユーザー認証フロー](diagrams/auth-sequence.md)
- [データ処理フロー](diagrams/data-processing.md)
- [API連携フロー](diagrams/api-integration.md)

### ガイド
システムのセットアップとデプロイに関する手順書です。

## クイックスタート

```bash
# 依存パッケージのインストール
pip install -r requirements.txt

# ローカルサーバーの起動
mkdocs serve

# ビルド
mkdocs build
```

!!! tip "ヒント"
    左側のナビゲーションメニューから各セクションにアクセスできます。
